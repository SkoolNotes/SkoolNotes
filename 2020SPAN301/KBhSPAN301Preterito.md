# Preterito
## Preterito Simple
**Estornudar** Sneeze

| S            | P            |
|--------------|--------------|
| Estornude    | Estornudamos |
| Estornudaste |              |
| Estornudó    | Estornudaron |

**Torser** Cough

| S        | P         |
|----------|-----------|
| Torsí    | Torsiamos |
| Torsiste |           |
| Torsió   | Torsieron |


* Estar - estu
* Poner - pu
* Querer - quis
* Tener - tuv

* Servir - servió
* Pedir - pidió
* Vestirse - vistió
* Divertirse - divertió

***

## Imperfecto

* Hacer => Hacía(s/n)
* Ir => Iba(s/n)
* Estar => Estaba(s/n)
* Ver => Veía(s/n)
* Ser => Era(s/n) 
* Llegar => Llega(s/n)
* Asistir => Asistía(s/n)
