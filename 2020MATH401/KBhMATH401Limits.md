# Limits
## Warming up
Here's a function

$y = \frac{1}{x}$. 

We know that it has

* Domain $D (-\infty, 0)(0, \infty)$
* Range $R (-\infty, 0)(0, \infty)$
* $As\ x\to\infty,\ y\to0$
* Function is *odd*, that is, $f(-x) = -f(x)$
