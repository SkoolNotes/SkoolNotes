# Group
- definition
	- closed
		- if $a, b \of S$ then $a + b \of S$
	- has an identity $e$
		- $e + a = a + e = a$
	- each element has an inverse
		- $-a + a = a + -a = e$
	- needs to be associative
		- $(a + b) + c$ = $a + (b + c)$
- communitivity is nice but not required
	- $a + b$ = $b + a$
